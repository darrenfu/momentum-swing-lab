# 01 - 先保护本金：风险、仓位和交易资格

Qullamaggie 的风格不是“高胜率慢慢赚”，而是用小亏损换少数大赢家。你要先学会亏小，再谈抓 5R、10R、20R 的机会。

> [!important] 第一原则
> 入场前必须知道：买点在哪里、错在哪里、亏多少、仓位多大、盈利后怎么处理。

## 1. 单笔风险

常见区间：

- 熟练交易者：单笔账户风险约 0.25% 到 1%。
- 新手训练期：建议从 0.10% 到 0.25% 的极小风险开始，甚至先只模拟。
- 高不确定性 setup：风险更小，或者直接跳过。

计算公式：

```text
账户风险金额 = 账户净值 * 单笔风险百分比
每股风险 = 入场价 - 止损价
可买股数 = 账户风险金额 / 每股风险
```

示例：

```text
账户净值: 100,000
单笔风险: 0.5% = 500
入场: 50
止损: 47.5
每股风险: 2.5
股数: 500 / 2.5 = 200
名义仓位: 200 * 50 = 10,000
```

## 2. 仓位上限

风险和仓位不是一回事。

- **风险**：止损触发时你预计亏多少。
- **仓位**：你买入股票的总市值。

Qullamaggie 的公开资料里常提到多数仓位大约 10% 到 20%，很多时候不希望单只股票隔夜超过账户 30%。新手不要从这里开始，而要先用仓位上限保护自己。

训练期建议：

- 单只股票名义仓位上限：5% 到 10%。
- 单笔风险：0.10% 到 0.25%。
- 同时持仓数：少于 5 个，方便复盘。

## 3. 止损必须在入场前写好

常见止损位置：

- Breakout：当日低点或形态失败点，且不应宽过 ATR/ADR 太多。
- EP：当日低点，或者开盘区间失败后的明确低点。
- Parabolic short：当日高点、VWAP reclaim，或 setup 失效点。

如果止损太远，正确做法不是扩大风险，而是降低仓位或跳过。

## 4. 不适合新手的东西

- Options、CFD、过度杠杆。
- 没有 hard stop 或 mental stop 训练却裸奔。
- 看别人直播就跟单。
- 因为错过 1 分钟 ORH，就在扩展后随便追。
- 不知道 earnings/catalyst 就只看形态。

## 5. 交易资格检查

每次实盘前问自己：

- 这是哪个 setup？
- 是否有足够样本支撑我理解它？
- 这只股票今天为什么动？
- 入场触发是否清楚？
- 止损在哪里？
- 如果止损成交，我亏账户多少百分比？
- 如果今天反向大幅 gap，我是否还能接受？
- 这笔交易是否在我的 focus list 里，还是临盘乱抓？

## 训练任务

- 建一个仓位计算表，用至少 20 个不同价格和止损距离练习 sizing。
- 找 10 张历史图，只做“如果我在这里入场，止损在哪里”的标注，不判断涨跌。
