# 07 - 从股票池到 Focus List

Qullamaggie/Deepvue 资源里的核心流程是把整个市场逐层缩小：Universe -> Wide List -> Weekly Focus List -> Daily Focus List。目的不是多看股票，而是让执行变简单。

## 1. Universe

Universe 是所有“可能值得看”的股票。规模可以是 300 到 600 个，或者按你的时间压到 200 到 300 个。

筛选维度：

- Relative strength。
- Liquidity：价格、成交量、average dollar volume。
- Trend：是否在关键均线上方。
- Fundamentals：EPS、sales、estimates。
- Industry/group strength。

## 2. Wide List

Wide List 是本周最值得跟踪的 50 到 100 个名字。

三类股票应进入 wide list：

- Leadership：当前市场最强、最符合你风格的 A+ 股票。
- Strong Movers：已经出现强走势，未来可能整理出 setup。
- Actionable Ideas：几天内可能触发交易计划的成熟形态。

## 3. Weekly Focus List

Weekly Focus List 是 5 到 20 个最高质量候选，越少越好。

用两个维度筛：

### Quality

- 它是不是 leader？
- 如果市场向好，它有没有翻倍潜力？
- Relative strength 是否足够强？
- 流动性是否足够让机构参与？
- 是否有高 short interest？
- 走势是否 tight，是否能 trend？
- EPS、sales、estimates 是否支持？
- 是否属于强行业或新主题？

### Readiness

- 接下来几天是否有可执行 entry？
- 是否接近完成更大 pattern？
- 最近 price action 是否 tight？
- 是否有即将发生的 catalyst？
- price/volume 是否 constructive？
- 是否接近 pivot？

## 4. Daily Focus List

Daily Focus List 最好少于 5 个。每天开盘前写清楚：

- 买点。
- stop。
- position size。
- 触发条件。
- 不交易条件。
- 如果 gap up/down 怎么处理。

## 5. 训练任务

每周末做一次：

1. 从全市场筛出 300 到 500 个 Universe。
2. 快速翻图，得到 50 到 100 个 Wide List。
3. 按 Quality 和 Readiness 打分，得到 5 到 20 个 Weekly Focus List。
4. 每天盘前从中选出少于 5 个 Daily Focus List。
5. 盘后记录：今天真正行动的是否来自 Focus List？
