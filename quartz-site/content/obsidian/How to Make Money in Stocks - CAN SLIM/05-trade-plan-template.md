## Sell Plan

### If wrong

- [ ] Exit if price hits stop.
- [ ] Exit if breakout fails and closes back below key level.
- [ ] Exit if market condition changes before setup works.

### If right

- [ ] Hold if price/volume action remains healthy.
- [ ] Consider adding only after first entry is profitable and a new valid buy point appears.
- [ ] Reduce or exit if price becomes extended, shows abnormal reversal, or violates key support.

## Pre-Trade Checklist

- [ ] I am not buying because the stock looks cheap.
- [ ] I am not chasing far above the pivot.
- [ ] I know exactly where I am wrong.
- [ ] Position size still lets me take the stop calmly.
- [ ] Market state supports the trade.
- [ ] This trade matches my current learning plan.
