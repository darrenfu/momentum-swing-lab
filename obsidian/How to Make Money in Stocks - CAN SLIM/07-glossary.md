# CAN SLIM Glossary

| Term | 中文 | 含义 | 用法 |
|---|---|---|---|
| CAN SLIM | 成长股七要素 | O'Neil 的选股和择时框架 | 用作候选股过滤，不是单一指标 |
| Base | 底部/整理形态 | 股价上涨前的整理结构 | 找买点前先判断 base 是否健康 |
| Pivot point | 枢轴买点 | 图形中突破阻力的关键价格 | 买入应尽量接近 pivot，而不是追太远 |
| Breakout | 突破 | 股价突破 base 或关键阻力 | 需要成交量和市场环境支持 |
| Cup with handle | 杯柄形态 | 常见成长股 base 之一 | 看深度、时间、右侧修复和 handle 质量 |
| Double bottom | 双底 | 两次下探后形成的 base | 第二个低点和突破位置要健康 |
| Flat base | 平台整理 | 强势股较浅的横盘整理 | 常出现在已有上涨后的再启动 |
| High tight flag | 高而紧的旗形 | 强势急涨后紧凑整理 | 罕见且风险较高，需要严格确认 |
| Faulty base | 有缺陷的底部 | 结构太明显、太松、波动异常或量价差 | 不要因为形态名字相似就买 |
| Volume dry-up | 成交量萎缩 | 整理期卖压减弱 | 常作为供给减少的证据 |
| Accumulation | 吸筹 | 上涨日放量，机构可能在买 | 观察价格上涨和成交量配合 |
| Distribution | 派发 | 下跌日放量，机构可能在卖 | 多个派发信号会提高市场风险 |
| Relative strength | 相对强度 | 股票相对大盘或同行的强弱 | 领导股通常先于大盘走强 |
| RS line | 相对强度线 | 个股价格除以指数或基准后的线 | 如果价格突破同时 RS 创高更好 |
| Leader | 领导股 | 行业或主题中最强的股票 | 优先研究龙头，而不是低位补涨 |
| Laggard | 落后股 | 同行业里表现弱的股票 | 通常不因“便宜”而买 |
| Follow-through day | 跟进日 | 市场下跌后尝试反转时的确认信号 | 用于判断大盘是否可能转强 |
| Buy zone | 买入区 | pivot 附近可接受的买入范围 | 超过太多就容易变成追高 |
| Extended | 过度延伸 | 股价离合理买点太远 | 再好也等下一次 base 或回踩 |
| 10-week line | 10 周均线 | 中期趋势参考线 | 强势股常在健康回踩中得到支撑 |
| Stop loss | 止损 | 预先定义的退出点 | 保护账户和心态的核心 |
| Average up | 向上加仓 | 盈利后在新买点加仓 | 与向下摊平相反 |
| Average down | 向下摊平 | 下跌后加仓降低成本 | CAN SLIM 系统中应避免 |
