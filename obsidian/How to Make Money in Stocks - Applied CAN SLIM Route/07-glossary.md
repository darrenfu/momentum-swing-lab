# CAN SLIM Glossary

Source: [Alternative Source](https://github.com/pistolla/gnidart/blob/master/How%20to%20Make%20Money%20in%20Stocks%20-%20A%20Winning%20System%20in%20Good%20Times%20and%20Bad%204th%20edition%202009.pdf)

| Term | 中文 | 用法 |
|---|---|---|
| CAN SLIM | 成长股交易系统 | 七个维度过滤候选股 |
| C - Current quarterly earnings | 当前季度盈利 | 最近季度 EPS 和销售是否强劲 |
| A - Annual earnings | 年度盈利 | 多年增长和盈利质量 |
| N - New | 新因素 | 新产品、服务、管理层、行业周期或新高 |
| S - Supply and demand | 供需 | 股本、成交量、突破和回调行为 |
| L - Leader or laggard | 领导股或落后股 | 买强者，避开弱者补涨幻想 |
| I - Institutional sponsorship | 机构支持 | 有机构需求，但警惕过度拥挤和派发 |
| M - Market direction | 市场方向 | 大盘是否支持做多 |
| Base | 底部/整理结构 | 股价消化供给、准备下一段趋势 |
| Cup with handle | 杯柄形态 | 常见高质量 base，需要 handle 和 pivot |
| Double bottom | 双底 | W 型整理，第二个低点和突破质量很重要 |
| Flat base | 平台整理 | 波动较窄的整理形态 |
| High tight flag | 高而紧的旗形 | 强势但稀有，要求非常严格 |
| Pivot point | 枢轴买点 | 突破点或可执行买点 |
| Breakout | 突破 | 从 base 中放量走出 |
| Volume dry-up | 成交量枯竭 | 回调中卖压减少 |
| Stalling | 滞涨 | 成交量大但价格推进不足，可能是派发 |
| Distribution | 派发 | 机构卖出压力，常见于大盘或个股顶部 |
| Relative strength | 相对强度 | 个股相对大盘或同行是否更强 |
| Climax top | 高潮顶部 | 短期极端上涨后的顶部风险 |
| Add-on buy | 加仓买点 | 第一笔盈利后，出现新的合理买点 |
| Averaging down | 向下摊平 | O'Neil 明确反对的行为 |
| Stop loss | 止损 | 控制小亏，保护组合 |
| Watchlist | 观察名单 | 候选股等待买点，不等于买入清单 |
| Actionable setup | 可执行形态 | 已接近买点且市场条件支持 |
